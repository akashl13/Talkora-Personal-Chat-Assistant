**Talkora** is a lightweight, desktop-based chatbot built using **Python** and **CustomTkinter**.  
It provides a real-time, interactive chat interface and can connect to the **Groq API** for intelligent responses.

## Features
- Modern GUI using CustomTkinter  
- Real-time chat interaction  
- Threading for smooth, non-blocking responses  
- Optional AI integration using Groq API  
- Simple, modular, and easy to extend  

## Requirements
- Python 3.10 or above  
- `customtkinter`  
- `openai`  
- `python-dotenv`

pip install customtkinter openai python-dotenv
